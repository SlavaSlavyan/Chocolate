# Chocolate

Небольшая программка которая выдаёт данные о продаже шоколада

Основная цель - это создать приложение которое сможет обрабатывать большие данные и выдавать какие то результаты на их основе. Для базовых данных я возьму продажи шоколада взятых с сайта [**Kaggle**](kaggle.com), как бесплатный источник, но теоретически, если данные будут иметь такую же структуру - то их тоже можно будет импортировать.

До первого релиза все изменения будут происходить в ветке **dev**!

## Как пользоваться программой

Вам понадобится файл БД с форматом **.csv**.

При запуске нажмите на кнопку **Загрузить CSV**. Если загрузка прошла успешно, то подготовка прошла успешно.

Нажимая на кнопки слева вам будут показа аналитика переданной информации.

### ВАЖНО!!!

Ваша таблица должна иметь правильные колонки для импорта.

- **Order_ID**
- **Product**
- **Country**
- **Channel**
- **Salesperson**
- **Order_Date**
- **Discount_Pct**
- **Price_per_Box**
- **Marketing_Spend**
- **Boxes_ShippedAmount**

## Требования

Язык программирования **python** версии **3.13**

Используемые библиотеки:

**БАЗОВЫЕ**

- **tkinter** - создание GUI

**СТОРОННИЕ**

- **matplotlib** - построение графиков

    - six
    - pyparsing
    - pillow
    - packaging
    - numpy
    - kiwisolver
    - fonttools
    - cycler
    - python-dateutil
    - contourpy

- **pandas** - работа с большими данными
    - tzdata

## Как запустить

Для запуска необходимо скачать все необходимые библиотеки из **Requirements**. Комманда для быстрой установки библиотек:

```bash
pip install matplotlib, pandas
```

Точкой входа является файл [**start.py**](start.py)
