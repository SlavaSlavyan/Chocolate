# Chocolate

Небольшая программка которая выдаёт данные о продаже шоколада

Основная цель - это создать приложение которое сможет обрабатывать большие данные и выдавать какие то результаты на их основе. Для базовых данных я возьму продажи шоколада взятых с сайта [**Kaggle**](kaggle.com), как бесплатный источник, но теоретически, если данные будут иметь такую же структуру - то их тоже можно будет импортировать.

До первого релиза все изменения будут происходить в ветке **dev**!

## Requirements

ЯП **python** версии **3.13**

Используемые библиотеки:

**БАЗОВЫЕ**

- **tkinter** - создание GUI

**СТОРОННИЕ**

- **matplotlib** - построение графиков

    - six
    - pyparsing
    - pillow
    - packaging
    - numpy
    - kiwisolver
    - fonttools
    - cycler
    - python-dateutil
    - contourpy,

## Как запустить

Для запуска необходимо скачать все необходимые библиотеки из **Requirements**. Комманда для быстрой установки библиотек:

```bash
pip install matplotlib
```
